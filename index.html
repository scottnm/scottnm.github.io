    Tools: Paint Tool Sai
  • Tinyrenderer

    Tools: C++
  • Squid Mantazord

  • Filibusters
    Fall 2016
    A multiplayer objective-based, arena-shooter game
    Tools: Unity, C#, Photon Unity Networking
  • Monkey Muck
    Fall 2016
    Designed, modeled, and textured an original 3D character. This character was inspired by horrific figures from franchises like Silent Hill, the Legend of Zelda, and the Thing.
    Tools: Maya, Photoshop
  • Pixitor
    Summer 2016
    A layer-based pixel art editor built using MIT"s Processing
    Tools: Processing, Java
  • Ogre JRPG
    Spring 2016
    A turn-based battle system demo using Ogre3D for one of my university courses.
    Tools: C++, Ogre3D, SDL
  • Labyrinth
    Fall 2015
    A game with procedurally generated maps, and a lighting effect where only a certain radius of the map away from the character is lit (a torch effect). Built for the 2015 Fall GameJam at UT Austin.
    Tools: C#, Unity, Gimp
  • Menger Sponge
    Fall 2015
    An OpenGL menger sponge that can be fully rotated and have the level of recursion updated on the fly via popup menus
    Tools: C++, OpenGL, GLUT
